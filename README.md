# 📝 Conta Bancária com JavaScript

Aplicação de simulador bancário desenvolvida com HTML, CSS e JavaScript puro, focada em Programação Orientada a Objetos (POO) e manipulação dinâmica de dados.

## 🚀 Funcionalidades

* **Depósitos e Saques:** Atualização de saldo em tempo real.
* **Transferências:** Envio de valores entre diferentes instâncias de contas (Ex: Pedro para Maria).
* **Histórico Dinâmico:** Registro das últimas 4 transações com horário detalhado.
* **Validação de Saldo:** Impede saques ou transferências maiores que o saldo disponível.
* **Status de Conta:** Alerta visual quando a conta está zerada.

## 🧠 Conceitos Praticados

* **Programação Orientada a Objetos (Classes e Métodos):** Organização da lógica de negócio.
* **Encapsulamento:** Uso de atributos privados (como `_saldo` e `_historico`).
* **Manipulação de DOM:** Criação dinâmica de elementos `<li>` para o histórico.
* **Lógica de Array:** Uso de `push()` e `shift()` para limitar o tamanho do histórico.
